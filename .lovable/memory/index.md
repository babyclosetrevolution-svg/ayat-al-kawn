# Project Memory

## Core
North Star: "Ne pas montrer l'Univers, mais faire ressentir son immensité." Every feature must reinforce immensity, contemplation, and progressive discovery — or be rejected.
Opening = Observer on Earth, looking up. Progression Terre → Système solaire → Voie lactée → Univers profond, sans rupture, par déplacement réel jamais par opacité.
Le vide est le décor, les astres sont des exceptions. Never dominate the frame with a single object (Sun, planets, galaxies).
Not a space game — a contemplative experience.
Vol = incarner un Observateur libre (caméra = Observateur), sensation Sangoku : fluide, silencieux, instinctif, sans secousses ; masse et contrôle préservés.
Ciel = vrai ciel (Soleil, Lune, planètes, étoiles réelles aux positions astronomiques). Jamais d'univers décoratif.
Monde continu : chaque astre visible est atteignable en vol, sans téléportation ni chargement visible.
